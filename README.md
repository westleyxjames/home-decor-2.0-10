# Luxe Nest Decor - HTML/CSS/JS Website

This directory contains the complete Luxe Nest Decor website converted from React to plain HTML, CSS, and JavaScript.

## Files Created

### Core Files
- **styles.css** - Main stylesheet with all CSS including responsive design
- **script.js** - JavaScript for all interactive features (mobile menu, forms, cookie consent, modals)
- **icons.svg** - SVG icon definitions (optional, inline SVGs are used in HTML)

### HTML Pages
- **index.html** - Home page with hero, services overview, testimonials, and pricing
- **about.html** - About Us page with company information and values
- **services.html** - Detailed services page with all 6 service offerings
- **contact.html** - Contact page with form, info, and Google Maps integration

### Legal Pages (All Completed ✅)
- **privacy-policy.html** - Privacy Policy including Google services disclosure
- **cookie-policy.html** - Cookie Policy with detailed Google cookies information
- **terms-conditions.html** - Terms & Conditions for using the services
- **disclaimer.html** - Legal disclaimer for services and information

## Features Included

### ✅ All Features Implemented
1. **Responsive Navigation** - Mobile menu with hamburger icon
2. **Lead Form Modal** - Popup form triggered by CTA buttons
3. **Cookie Consent Banner** - Dismissible banner with localStorage
4. **Contact Forms** - With policy consent checkboxes
5. **Google Maps Integration** - Embedded map on contact page
6. **Premium Styling** - Warm neutral colors (beige, cream, gold accents)
7. **Smooth Animations** - CSS transitions and hover effects
8. **Google Policies** - Comprehensive Google Maps and Analytics disclosures

### JavaScript Functionality
- Mobile menu toggle
- Active navigation highlighting
- Cookie consent management (localStorage)
- Lead form modal open/close
- Contact form handling
- Form validation
- Smooth scrolling for anchor links

## How to Use

1. **Open index.html** in a web browser to view the website
2. All pages are fully self-contained with inline styles references
3. No build process required - pure HTML/CSS/JS
4. Forms use client-side validation and console logging

## Browser Compatibility
- Chrome/Edge (modern versions)
- Firefox (modern versions)
- Safari (modern versions)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Hosting
To host this website:
1. Upload all files in the `/public` directory to your web server
2. Ensure index.html is set as the default page
3. All links are relative and will work without modification
4. No server-side processing required

## Notes
- All images are loaded from Unsplash CDN
- Google Maps embed uses a placeholder URL (update with your actual map embed URL)
- Form submissions are logged to console (integrate with backend as needed)
- localStorage is used for cookie consent persistence

## Maintenance
- Update the "Last updated" date on legal pages when content changes
- Keep Google Maps API disclosure current if you add Analytics tracking
- Test forms regularly to ensure they work correctly
- Check mobile responsiveness after any CSS changes
