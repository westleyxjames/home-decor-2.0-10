### Option 2: Web Hosting

2. Ensure `index.html` is set as the default/index page
3. All relative links will work automatically

2. Set `index.html` as the entry point

## 🔧 Customization Guide

### Update Contact Information
Edit these sections in all HTML files:
- Footer contact details
- Contact page information
- Legal pages contact sections

### Update Google Maps
In `contact.html`, replace the iframe src with your actual Google Maps embed URL:
```html
<iframe src="YOUR_GOOGLE_MAPS_EMBED_URL" ...>
```

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
  --color-primary: #8B7355;
  --color-gold: #D4AF37;
  /* etc. */
}
```

### Add Backend Integration
Forms currently log to console. To integrate with a backend:
1. Update form submission handlers in `script.js`
2. Add AJAX/fetch calls to your API endpoint
3. Handle responses and display success/error messages
